# firebase_analytics_module
help to use firebase_analytics in june_flow framework

##  Installation
1. If the juneflow project doesn't exist, please create it by following [this guide](https://doc.juneflow.org/).
2. open terminal in the juneflow project root directory, enter the following command.
 ```bash
 june add firebase_analytics_module
 ```

## Usage
more detail usage about firebase_analytics, please refer to [firebase_analytics](https://pub.dev/packages/firebase_analytics)
